# MyFriend C2C E-Commerce Platform

MyFriend is a PHP and MySQL customer-to-customer e-commerce platform designed for South African informal traders and buyers.

## Features

- User registration and login
- Forgot-password functionality
- Buyer, seller and administrator roles
- Product marketplace
- Search, category and province filters
- Seller shops and contact details
- Shopping cart
- Online demonstration payments
- Cash on delivery and cash on collection
- Delivery and collection options
- Buyer order tracking
- Seller order management
- Complaints and administrator management
- Responsive Bootstrap interface

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- MySQL/MariaDB
- phpMyAdmin
- XAMPP
- InfinityFree

## Local Setup

1. Copy the project files into `C:\xampp\htdocs`.
2. Start Apache and MySQL in XAMPP.
3. Import `database/database_schema.sql` into phpMyAdmin.
4. Copy `config/database.example.php` to `config/database.php`.
5. Add the local database credentials.
6. Open `http://localhost/`.
